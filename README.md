# Карта с маркерами - React Native приложение

Мобильное приложение для создания маркеров на карте с возможностью прикрепления изображений, использованием базы данных sqlite, а также отслеживанием геолокации

## Функциональность

* Добавление маркеров долгим нажатием на карту
* Выбор цвета маркера
* Добавление названия и описания к маркерам
* Прикрепление изображений к маркерам
* Просмотр изображений в полноэкранном режиме
* Удаление маркеров и изображений
* Отслеживание геолокации
* Показ уведомлений
* Возможность центрировать камеру на пользователе при включенной геолокации

## Технологии

* React Native
* Expo
* TypeScript
* React Native Maps
* Expo Image Picker
* Expo Router
* Expo sqlite
* Expo notifications
* Expo location

## Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone <URL_репозитория>
cd <папка_проекта>
```

2. Установите зависимости:

```bash
npm install
```

3. Запустите приложение:

```bash
npx expo start
```

4. Откройте приложение Expo Go на вашем устройстве и отсканируйте QR-код, чтобы протестировать приложение

5. Если потребуется, можно использовать мокирование геолокации, для этого рекомендую использовать Listick Fake GPS (Android), в нём можно менять позицию с помощью джойстика с заданной скоростью

## Дополнительно реализованные функции

* Выбор цвета маркера из 5 возможных в модальном окне при долгом нажатии
* Заполнения названия и описания маркера в модальном окне при долгом нажатии
* Галерея в деталях маркера сделана под мобильный стиль (плитки)
* Удаление маркера со страницы деталей
* Возможность перехода из уведомления в маркер
* Центрирование камеры на пользователе при нажатии на кнопку с булавкой
* Вывод координат пользователя и кол-ва показанных уведомлений

## Проблемы и ограничения

* Данные маркеров и изображений сохраняются только в памяти приложения и будут сброшены после перезагрузки
* Для полноценного использования всех функций требуется разрешение на доступ к галерее для добавления изображений
* Можно выбирать только по 1 изображению за раз
* Карта доступна только для мобильных устройств
* При запуске пишется сначала предупреждение, а потом ошибка, что для Expo go не до конца доступны все функции уведомлений, но при этом локальные уведомления работают нормально
